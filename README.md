# Balancing the Word Ministry Website

This is the live website project. GitHub Pages publishes this folder.

## Use These Folders

- `index.html` - the homepage. Keep this at the root.
- `pages/` - regular website pages such as About, Books, Support, Podcast, and the Article + Audio archive page.
- `articles/` - the current featured article page. Replace this when a new featured article goes live.
- `archive/` - full older article pages. Visitors can still open and read these.
- `Assets/` - active images, audio, downloads, and layout files used by the website.

## Usually Leave These Alone

- `CNAME` - required for the balancingtheword.org domain.
- `.git/` and `.gitignore` - website version history files.
- Root files such as `about.html`, `books.html`, and `building-gods-kingdom.html` may be older public links. Do not delete them unless you intentionally replace them with redirects.
- Older `images/`, `logos/`, and `blog/` folders were removed after their references were cleaned up. Active media now lives under `Assets/`.

## Notes And Templates

Non-public workflow notes, templates, and reference files live in `_workspace-notes/`.
